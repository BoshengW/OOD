### 虚拟钱包支付设计
- 需求
  - 开设一个虚拟钱包账户
    - 虚拟钱包账户通常绑定银行账户 - 其本身只具备充值，提现，支付，查询余额
  - 支持用户充值
    - 流程: 绑定银行账号 -> 虚拟钱包公司公共银行账号 -> 给虚拟钱包添加金额 -> 记录流水
  - 支持提现
    - 流程：公司公共银行账户 -> 绑定银行账户 -> 虚拟钱包中扣钱 -> 记录流水
  - 支付功能
    - 流程：虚拟钱包中划钱 -> 商家虚拟钱包
  - 冻结
  - 透支
  - 转赠
    - 流程: 虚拟钱包 -> 另一个人虚拟钱包
  - 查询账户余额
  - 查询交易流水
